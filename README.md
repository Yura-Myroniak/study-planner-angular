# Study Planner (Angular)

## Тема проєкту
Study Planner — навчальний SPA-додаток на Angular для керування предметами та завданнями з дедлайнами.
Дані реалізовані без реального Back-end: імітація через Angular Services (mock data).

##  Автори
- Мироняк Юрій (Yura-Myroniak)
- Качур Віктор (flyw1x)

##  Функціонал
- Список предметів + додавання предмета
- Список завдань + фільтрація за статусом (pipe)
- Сторінка деталей предмета
- Routing між сторінками
- 404 сторінка

##  Використано (Angular basics)
- Components
- Routing
- Services (mock back-end)
- Template-driven forms (ngModel)
- Custom Pipe
- 404 page
